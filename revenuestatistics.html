Revenue Statistics User Guide

This guide explains how to use the **Revenue Statistics** feature in the HK.Huang Sales Management Software, allowing you to view and filter sales data by date range to generate revenue reports.

Overview of the Revenue Statistics Screen

The Revenue Statistics screen (frmRevenueStatistics) provides a comprehensive view of your sales revenue. You can analyze total revenue for specific periods by applying date filters.

Image: The Revenue Statistics interface with date filter options and report viewer.

Initial Load and Default Report

When you first open the Revenue Statistics screen (rptRevenueStatistics_Load):

  • The report viewer automatically displays a comprehensive list of All Orders (and thus, all revenue) for all time.
  • The report includes details such as Order ID, Employee Name, Customer Name, Date, Note, and Total Price.
  • The report's title will indicate "(All time)" as the current filter description.

Filtering Revenue by Date Range

You can view revenue for a specific period by setting a date range:

  1. Select Start Date: Use the Start Date date picker (dtpStart) to choose the beginning date of your desired period.
  2. Select End Date: Use the End Date date picker (dtpEnd) to choose the ending date of your desired period.
  3. Apply Filter: Click the Filter button (btnFilter_Click).
    • The report viewer will refresh to show only orders (and revenue) within your selected date range.
    • The report's title will update to reflect the applied date range (e.g., "From date: MM/dd/yyyy - To date: MM/dd/yyyy").
Note: Ensure your Start Date is not after your End Date for meaningful results.

Viewing All Time Revenue

To revert the report to show all orders and revenue from all time, regardless of any previously applied filters:

  • Click the Show All button (btnShowAll_Click). The report will reset to its default "All time" view.

Printing/Exporting Statistics to PDF

To save the currently displayed revenue statistics report as a PDF file:

  1. Click the Print button (btnPrint_Click).
  2. A "Save As" dialog box will appear.
  3. Choose a location on your computer to save the file.
  4. The default file name will be "Revenue_Statistics_DD_MM_YYYY.pdf" (e.g., `Revenue_Statistics_04_07_2025.pdf`). You can change this if desired.
  5. Click Save.
  6. A "Revenue Statistics saved successfully!" message will confirm that the PDF has been generated and saved.
Warning: If there is no data in the report viewer (e.g., after applying filters that yield no results), a warning message "No report data to save." will appear when attempting to print.

Closing the Revenue Statistics Screen

  • Click the Close button (btnClose_Click) to exit the Revenue Statistics screen.
