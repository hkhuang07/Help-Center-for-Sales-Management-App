Exit Program User Guide

This guide explains how to safely and properly exit the **HK.Huang Sales Management Software**.

Overview of Exiting the Program

Exiting the program closes all active windows and ensures that all data is saved properly (if applicable) before shutting down the application.

Image: A typical menu location for exiting or logging out of the application.

How to Exit the Program

There are several common ways to exit the HK.Huang Sales Management Software:

  1. Using the Application Menu:
    • Navigate to the main menu of the application, usually labeled System, File, or a similar heading.
    • Look for an option such as Exit, Close Application, or Logout (as shown in the example image, if "Logout" is configured to close the program).
    • Click on this option to initiate the exit process.
  2. Using the Close Button:
    • Click the standard X button located in the top-right corner of the application's main window. This is the universal close button for most Windows applications.
  3. Using Keyboard Shortcut:
    • Press Alt + F4 on your keyboard while the application's main window is active. This is a common Windows shortcut to close the current window.

Important Considerations Before Exiting

Always save your work: Before exiting, ensure that you have saved any unsaved changes in any open forms or documents within the application. The system may prompt you to save unsaved work, but it's always best practice to save proactively.
Confirmation Dialog: Depending on the application's configuration, you might encounter a confirmation dialog asking "Are you sure you want to exit?" or "Do you want to save changes before exiting?". Always review these prompts and respond appropriately.
