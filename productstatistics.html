Product Statistics User Guide

This guide explains how to use the **Product Statistics** feature in the HK.Huang Sales Management Software, allowing you to view and filter product data and generate reports.

Overview of the Product Statistics Screen

The Product Statistics screen (frmProductStatistics) is designed to provide detailed insights into your products, including their categories, manufacturers, quantities, and prices. You can filter this data to generate specific reports.

Image: The Product Statistics interface with filter options and report viewer.

Initial Load and Default Report

When you first open the Product Statistics screen (frmProductStatistics_Load):

  • The Category and Manufacturer dropdowns (cboCategory, cboManufacturer) are populated with available data.
  • The report viewer automatically displays a comprehensive list of All Products, including their ID, Manufacturer, Category, Product Name, Price, Quantity, Image (if applicable), and Description.
  • The report's title will indicate "(All Products)" as the current filter description.

Filtering Product Statistics

You can narrow down the product list displayed in the report viewer by applying filters:

  1. Select Category: Choose a specific category from the Category dropdown.
  2. Select Manufacturer: Choose a specific manufacturer from the Manufacturer dropdown.
  3. Apply Filter: Click the Filter button (btnFilter_Click).
    • The report viewer will refresh to show only products matching your selected criteria.
    • The report's title will update to reflect the applied filters (e.g., "(Manufacturer: [Selected Manufacturer] - Category: [Selected Category])").
  4. Clear Filters: If both Category and Manufacturer dropdowns are empty, clicking the Filter button will reload the report to show all products again. You can manually clear the dropdowns to achieve this.
Tip: You can apply filters individually (e.g., filter by Category only, or by Manufacturer only) or combine them.

Printing/Exporting Statistics to PDF

To save the currently displayed product statistics report as a PDF file:

  1. Click the Print button (btnPrint_Click).
  2. A "Save As" dialog box will appear.
  3. Choose a location on your computer to save the file.
  4. The default file name will be "Product_Statistics_DD_MM_YYYY.pdf" (e.g., `Product_Statistics_04_07_2025.pdf`). You can change this if desired.
  5. Click Save.
  6. A "Product statistics saved successfully!" message will confirm that the PDF has been generated and saved.
Note: If there is no data in the report viewer (e.g., after applying filters that yield no results), a warning message "No report data to save." will appear when attempting to print.

Closing the Product Statistics Screen

  • Click the Close button (btnClose_Click) to exit the Product Statistics screen.
