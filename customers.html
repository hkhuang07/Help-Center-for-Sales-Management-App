Customer Management User Guide

This guide explains how to manage customer information within the **HK.Huang Sales Management Software**. Accurate customer data is vital for sales, marketing, and customer service.

Overview of the Customers Screen

The Customers management screen (frmCustomers) allows you to view, add, edit, delete, import, and export customer details. The main components are:

  • Customer List (Data Grid View): Displays all existing customers with their details including ID, Name, Address, Phone, and Email.
  • Customer Details Panel: Shows the details of the currently selected customer for editing or new entry. This includes Customer Name, Address, Phone, and Email.
  • Action Buttons: Buttons for various operations like Add, Update, Delete, Save, Cancel, Import, and Export.
  • Navigation and Search Bar: Tools to navigate through the list and find specific customers.

Image: The main interface for managing customers.

Viewing and Navigating Customers

When you open the Customers screen (Customers_Load), the system automatically loads all available customer records into the data grid view (dataGridView).

1. Navigating Through Records

Use the navigation buttons on the toolbar to move through the list of customers:

  • First (btnBegin): Moves to the first record.
  • Previous (btnPrevious): Moves to the previous record.
  • Next (btnNext): Moves to the next record.
  • Last (btnEnd): Moves to the last record.

The details of the currently selected customer will be displayed in the respective fields: "Customer Name" (txtCustomerName), "Address" (txtCustomerAddress), "Phone" (txtCustomerPhone), and "Email" (txtCustomerEmail).

2. Searching for Customers

To find a specific customer:

  • Enter keywords into the search box (txtFind).
  • As you type, the list will automatically filter to show customers whose Name, Address, Phone, or Email contain the entered keyword (triggered by txtFind.TextChanged calling btnFind.PerformClick).
  • If no matching customers are found, a "No matching customer found." message will appear (in lblMessage).
  • Click the Clear button (btnClear_Click) to clear the search filter and display all customers again.

Adding a New Customer

To add a new customer record:

  1. Click the Add button (btnAdd_Click). All customer detail fields will become active and clear for new input, and the "Save" and "Cancel" buttons will be enabled.
  2. Enter the details for the new customer into the respective fields:
    • Customer Name (txtCustomerName)
    • Customer Address (txtCustomerAddress)
    • Customer Phone (txtCustomerPhone)
    • Customer Email (txtCustomerEmail)
  3. Click the Save button (btnSave_Click) to create the new customer.
  4. A "Operation successful!" message will confirm the addition. The screen will then refresh, and the new customer will appear in the list.
Note: Ensure all required fields are filled accurately.

Editing an Existing Customer

To modify an existing customer record:

  1. Select the customer you wish to edit from the data grid view (dataGridView) by clicking on its row. The customer's details will appear in the input fields.
  2. Click the Update button (btnUpdate_Click). All customer detail fields will become editable, and the "Save" and "Cancel" buttons will be enabled.
  3. Modify the customer details as needed in the txtCustomerName, txtCustomerAddress, txtCustomerPhone, and txtCustomerEmail fields.
  4. Click the Save button (btnSave_Click) to apply your changes.
  5. A "Operation successful!" message will confirm the update. The screen will then refresh to reflect the changes.

Deleting a Customer

To remove a customer record:

  1. Select the customer you wish to delete from the data grid view (dataGridView).
  2. Click the Delete button (btnDelete_Click).
  3. A confirmation message "Are you sure you want to delete this customer?" will appear. Click Yes to proceed or No to cancel.
  4. If confirmed, a "Customer deleted successfully!" message will appear, and the customer will be removed from the list.
Warning: Deleting a customer will permanently remove it from the system.

Importing Customers from Excel

You can import multiple customer records from an Excel file:

  1. Click the Import button (btnImport_Click). An "Open File" dialog will appear.
  2. Navigate to and select your Excel file (e.g., .xls or .xlsx).
  3. The system will read the data from the first worksheet. Ensure your Excel file has header columns named exactly: "CustomerName", "CustomerAddress", "CustomerPhone", and "CustomerEmail".
  4. After the import process, a message indicating how many rows were successfully imported will appear.
Excel File Format:
The Excel file must have a header row with the exact column names: "CustomerName", "CustomerAddress", "CustomerPhone", and "CustomerEmail".

Exporting Customers to Excel

You can export the current list of customers to an Excel file:

  1. Click the Export button (btnExport_Click). A "Save File" dialog will appear.
  2. Choose a location and enter a file name for your Excel file (e.g., Customers_YYYY_MM_DD.xlsx).
  3. Click Save.
  4. A "Exported data to Excel file successfully." message will confirm the export.
  5. The Excel file will contain columns for ID, Customer Name, Address, Phone, and Email.

Cancelling Actions and Closing

  • Cancel Button (btnCancel_Click): If you are in "Add" or "Update" mode, clicking "Cancel" will discard any unsaved changes and return the screen to its initial state, reloading the original data.
  • Close Button (btnClose_Click): Closes the Customers management screen and returns you to the main application.
