Flash Screen User Guide

This guide explains the purpose and behavior of the **Flash Screen** (màn hình khởi động) that appears when you launch the HK.Huang Sales Management Software.

Overview of the Flash Screen

The Flash screen (Flash Form) is a brief introductory display that appears automatically when you start the application. Its primary purpose is to provide a visual cue that the software is loading and preparing for use.

Image: The Flash screen with a logo and progress bar.

Functionality and Duration

  • Automatic Appearance: The Flash screen appears automatically as soon as you launch the HK.Huang Sales Management Software. You do not need to click anything to open it.
  • Loading Indicator: The screen typically displays a logo or application name and a progress bar at the bottom. This progress bar visually indicates that the application is initializing.
  • Temporary Display: The Flash screen is designed to be temporary. It will display for a few seconds (approximately 3 seconds as configured) while background processes complete.
  • Automatic Closing: Once the loading process is complete, the Flash screen will automatically close (this.Close()) and the main application window will appear. You do not need to manually interact with this screen to close it.

User Interaction

There is no user interaction required or possible with the Flash screen. It serves purely as an informational and loading indicator. You cannot click on elements, enter data, or manually close this screen.
