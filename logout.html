Logout Function User Guide

This section provides a comprehensive guide on how to safely and effectively **Log Out** from the **HK.Huang Sales Management Software**. Logging out ensures the security of your data and frees up system resources when you are finished with your session.

Understanding the Logout Process

The logout process involves closing your current session within the application. This action is crucial for security, especially if you are using a shared computer or stepping away from your workstation. It resets the application to a pre-login state, ready for another user to log in or for the application to be exited.

How to Log Out

Follow these steps to log out of the system:

1. Accessing the Logout Option

The logout option is typically found within the application's main menu or toolbar. Look for an icon or text indicating "Logout" or "Sign Out".

Image: Location of the 'Logout' option (e.g., in a menu).

Click on the Logout menu item (corresponding to the mnuLogout_Click event in the C# code).

2. Confirming Your Action

Upon attempting to log out, the system will prompt you with a confirmation message to prevent accidental logouts. This is a standard security measure.

Image: The logout confirmation dialog box.

  • If you wish to proceed with logging out, click the Yes button.
  • If you wish to cancel the logout and continue using the application, click the No button.

3. Post-Logout State

If you confirm the logout (by clicking "Yes"), the application will:

  • Close all currently open child windows within the main application interface (as handled by the `foreach (Form child in MdiChildren) { child.Close(); }` loop in mnuLogout_Click).
  • Reset the application's user interface to its "Not Logged In" state (by calling the NotLoggedIn() method). This means most functionalities will be disabled, and the system will be ready for a new user to log in.
  • The status bar (if available) will display "Not Logged In." (as set by lblStatus.Text = "Not Logged In.";).
Important Note:
Always remember to log out when you are finished using the software, especially if you are on a shared computer, to protect sensitive data and maintain system security.
