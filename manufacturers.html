Product Manufacturers Management User Guide

This guide explains how to manage manufacturer information within the **HK.Huang Sales Management Software**. Effective manufacturer management helps in tracking product origins and supplier details.

Overview of the Manufacturers Screen

The Manufacturers management screen (frmManufacturers) allows you to view, add, edit, delete, import, and export manufacturer details. The main components are:

  • Manufacturer List (Data Grid View): Displays all existing manufacturers with their ID, Name, Address, Phone, and Email.
  • Manufacturer Details Panel: Shows the name, address, phone, and email of the currently selected manufacturer for editing or new entry.
  • Action Buttons: Buttons for various operations like Add, Edit, Delete, Save, Cancel, Import, Export.
  • Navigation and Search Bar: Tools to navigate through the list and find specific manufacturers.

Image: The main interface for managing product manufacturers.

Viewing and Navigating Manufacturers

When you open the Manufacturers screen (Manufacturer_Load), the system automatically loads all available manufacturer records into the data grid view (dataGridView).

1. Navigating Through Records

Use the navigation buttons on the toolbar to move through the list of manufacturers:

  • First (btnBegin): Moves to the first record.
  • Previous (btnPrevious): Moves to the previous record.
  • Next (btnNext): Moves to the next record.
  • Last (btnEnd): Moves to the last record.

The details of the currently selected manufacturer will be displayed in the respective fields: "Manufacturer Name" (txtManufacturerName), "Manufacturer Address" (txtManufacturerAddress), "Manufacturer Phone" (txtManufacturerPhone), and "Manufacturer Email" (txtManufacturerEmail).

2. Searching for Manufacturers

To find a specific manufacturer:

  • Enter keywords into the search box (txtFind).
  • As you type, the list will automatically filter to show manufacturers whose Name, Address, Phone, or Email contain the entered keyword (triggered by txtFind_TextChanged calling btnFind_Click).
  • If no matching manufacturers are found, a "No matching customer found." message will appear (in lblMessage).
  • Click the Clear button (btnClear_Click) to clear the search filter and display all manufacturers again.

Adding a New Manufacturer

To add a new manufacturer record:

  1. Click the Add button (btnAdd_Click). All manufacturer detail fields will become active and clear for new input, and the "Save" and "Cancel" buttons will be enabled.
  2. Enter the details for the new manufacturer into the respective fields:
    • Manufacturer Name (txtManufacturerName)
    • Manufacturer Address (txtManufacturerAddress)
    • Manufacturer Phone (txtManufacturerPhone)
    • Manufacturer Email (txtManufacturerEmail)
  3. Click the Save button (btnSave_Click) to create the new manufacturer.
  4. A "Operation successful!" message will confirm the addition. The screen will then refresh, and the new manufacturer will appear in the list.
Note: Ensure all required fields are filled accurately.

Editing an Existing Manufacturer

To modify an existing manufacturer record:

  1. Select the manufacturer you wish to edit from the data grid view (dataGridView) by clicking on its row. The manufacturer's details will appear in the input fields.
  2. Click the Edit button (btnEdit_Click). All manufacturer detail fields will become editable, and the "Save" and "Cancel" buttons will be enabled.
  3. Modify the manufacturer details as needed in the txtManufacturerName, txtManufacturerAddress, txtManufacturerPhone, and txtManufacturerEmail fields.
  4. Click the Save button (btnSave_Click) to apply your changes.
  5. A "Operation successful!" message will confirm the update. The screen will then refresh to reflect the changes.

Deleting a Manufacturer

To remove a manufacturer record:

  1. Select the manufacturer you wish to delete from the data grid view (dataGridView).
  2. Click the Delete button (btnDelete_Click).
  3. A confirmation message "Are you sure you want to delete this manufacturer?" will appear. Click Yes to proceed or No to cancel.
  4. If confirmed, a "Manufacturer deleted successfully!" message will appear, and the manufacturer will be removed from the list.
Warning:
Deleting a manufacturer that is linked to existing products may cause data integrity issues or errors. Ensure no products are associated with the manufacturer before deleting it.

Importing Manufacturers from Excel

You can import multiple manufacturer records from an Excel file:

  1. Click the Import button (btnImport_Click). An "Open File" dialog will appear.
  2. Navigate to and select your Excel file (e.g., .xls or .xlsx).
  3. The system will read the data from the first worksheet. Ensure your Excel file has header columns named exactly: "ManufacturerName", "ManufacturerAddress", "ManufacturerPhone", and "ManufacturerEmail".
  4. After the import process, a message indicating how many rows were successfully imported will appear.
Excel File Format:
The Excel file must have a header row with the exact column names: "ManufacturerName", "ManufacturerAddress", "ManufacturerPhone", "ManufacturerEmail".

Exporting Manufacturers to Excel

You can export the current list of manufacturers to an Excel file:

  1. Click the Export button (btnExport_Click). A "Save File" dialog will appear.
  2. Choose a location and enter a file name for your Excel file (e.g., Manufacturers_YYYY_MM_DD.xlsx).
  3. Click Save.
  4. A "Exported data to Excel file successfully." message will confirm the export.
  5. The Excel file will contain columns for ID, Manufacturer Name, Email, Phone, and Address.

Cancelling Actions and Closing

  • Cancel Button (btnCancel_Click): If you are in "Add" or "Edit" mode, clicking "Cancel" will discard any unsaved changes and return the screen to its initial state.
  • Close Button (btnClose_Click): Closes the Manufacturers management screen and returns you to the main application.
